## Основи керування віртуальними ОС

**Лабораторна робота №5**
**Виконав:** Дорошков Родіон
**Група:** АІ-253

### 1. Підготовка середовища (Розділ 2.1 - 2.2)
- Створено скрипт автоматизації `CreateVMDoroshkov.sh` для розгортання ВМ через CLI `VBoxManage`.
- Налаштовано прокидання портів для доступу по SSH (порт 1234).
- Встановлено та налаштовано Git всередині гостьової ОС Ubuntu.
- Сгенеровано SSH-ключі для авторизації на GitHub та клоновано репозиторій.

### 2. Керування користувачами та групами (Розділ 2.3)
- Створено системну групу `ai253`.
- Додано нового користувача `doroshkov_rodion` з власною домашньою директорією та оболонкою `/bin/bash`.
- Встановлено пароль для нового користувача.
- Проведено тест взаємодії між терміналами:
  - Використано команду `who` для ідентифікації активних сесій (`pts/1`, `pts/2`).
  - Виконано успішну передачу повідомлення від `doroshkov_rodion` до `doroshkov` за допомогою утиліти `write`.

### 3. Результати роботи з Git (Розділ 2.4)
- Створено локальну гілку `Laboratory-work-5`.
- Всі зміни зафіксовано (commit) та відправлено (push) до віддаленого репозиторію.

### Висновок
В ході виконання роботи було опановано методи керування віртуальними машинами через командний рядок, принципи адміністрування користувачів у Linux та взаємодію між процесами через термінали. Робота з Git дозволила закріпити навички версіонування коду та синхронізації локальних змін з хмарним сервісом GitHub.
